# Centaur reference experiments

These scripts are retained as research references for the planned head-to-head
comparison with Centaur. They are not part of the supported `mt` API and have
not yet been reviewed against the current Transformers/TRL versions.

Do not import helpers from this directory into `src/mt`. When the comparison
protocol is designed, use these files as evidence about the old workflow and
replace them with a tested experiment entry point.
